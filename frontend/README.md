# 🌍 ResQAI – AI Smart Disaster Assistant
## 📌 Overview
ResQAI is an AI-powered disaster management web application developed using React.js. It provides users with real-time weather, disaster news, earthquake monitoring, AI-based emergency guidance, and nearby emergency services to improve disaster preparedness and response.
---
## ✨ Features
- 🌤 Live Weather Information
- 🤖 AI Emergency Assistant (Google Gemini)
- 🌍 Live Disaster Map
- 🌎 Live Earthquake Alerts
- 📰 Live Disaster News
- 📍 Current GPS Location
- 🚨 Emergency SOS
- 🏥 Nearby Hospitals
- 👮 Nearby Police Stations
- 🚒 Nearby Fire Stations
- 📞 Emergency Contact Numbers
---
## 🛠 Technologies Used
